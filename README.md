# Matrix Bot Client

A lightweight Matrix PWA designed for interacting with bots. Works with any Matrix bot setup.

## Features

- Matrix login (username/password)
- Dynamic room list fetched from your homeserver
- Per-room pill shortcuts for common questions
- As-you-type autocomplete/suggestions
- Installable as a PWA (iOS: Add to Home Screen, Android/desktop: install prompt)
- No room management, no bloat — just messaging

## Getting started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
